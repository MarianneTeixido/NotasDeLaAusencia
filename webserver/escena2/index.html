• Ayuda

  • Haz clic en pantalla y desplaza el mouse para mover la cámara.
  • Utiliza el scroll del mouse para acercarte o alejarte de los objetos
  • Presiona 1 para ordenar las palabras de menor a mayor recurrencia.
  • Presiona 2 para ordenar las palabras en aleatorio.
  • Desplaza el cursor sobre las esferas para visualizar la palabras.
  • Presiona m para navegar en las escenas.
